# Titanic Dataset – Exploratory Data Analysis

## Task Description
This project is part of SkillCraft Technology Internship Task 02.
The objective is to perform data cleaning and exploratory data analysis
on the Titanic dataset to identify patterns and trends.

## Dataset
- Titanic Dataset (Kaggle)

## Tools Used
- Python
- Jupyter Notebook
- Pandas, NumPy
- Matplotlib, Seaborn

## Steps Performed
- Loaded and explored the dataset
- Handled missing values
- Removed unnecessary columns
- Performed exploratory data analysis using visualizations

## Key Insights
- Female passengers had higher survival rates
- Passengers in 1st class survived more
- Higher fare passengers had better survival chances
- Most passengers were aged between 20–40

- Titanic_EDA.ipynb :jupyter notebook containing data cleaning and EDA
- Titanic-Dataset.csv:Dataset used for analysis
